== Description ==

This plugin will help customers to upload an attachment during the woocommerce checkout. Later shop owners will approve/ reject the order based on attachment. Attachment can be images/pdf/docs containing medical description,design sample,sample items etc.

**Free Version Features:**

* Enable order approval with attachment globally or per product-wise.

* No payment will be taken during checkout.
* Shop owners can approve/cancel orders.

* Approved order will get a payment link to proceed.

* Showing attachment in the order edit page.
* **Support JPG/JPEG Images Files with Size less than 1MB**

**Premium Version Features**

* All featured of FREE version.
* Support 11 file types (Jpeg/Jpg, Png, Gif,Pdf,Doc,Docx, Ppt,Pptx,Xls,Xlsx,Csv etc.. )
* Custom maximum file upload size.

== Installation ==

1. Upload `sg-woocommerce-order-approval-with-attachment.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
